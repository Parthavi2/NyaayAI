📌  Overview
NyaayAI is a full-stack AI legal aid platform that empowers individuals who cannot afford legal counsel. Users can upload legal documents (PDFs, images, scans), and the system uses large language models to extract key clauses, identify risks, summarise obligations, and suggest next steps — all in plain language.

The platform is built on a Next.js 14 frontend and a FastAPI backend, with OCR support for scanned documents, JWT authentication, and persistent storage via SQLite (dev) or PostgreSQL (production).

💻  Local Setup
Prerequisites
•	Python 3.10+
•	Node.js 18+
•	Tesseract OCR  (brew install tesseract  |  sudo apt install tesseract-ocr)
•	Groq API key from console.groq.com (free)

Backend
cd backend
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python -m spacy download en_core_web_sm

# Create .env
echo 'GROQ_API_KEY=gsk_your_key_here' > .env
echo 'SECRET_KEY=change-me-in-production' >> .env
echo 'DATABASE_URL=sqlite:///./nyaay_ai.db' >> .env

uvicorn app.main:app --reload --port 8000

Frontend
cd frontend
npm install
echo 'NEXT_PUBLIC_API_URL=http://localhost:8000' > .env.local
echo 'NEXT_PUBLIC_USE_MOCK=false' >> .env.local
npm run dev
Frontend runs at http://localhost:3000   |   API docs at http://localhost:8000/docs

☁️  Deployment Guide
Backend → Render
Root Directory	backend
Build Command	pip install -r requirements.txt
Start Command	uvicorn app.main:app --host 0.0.0.0 --port $PORT
GROQ_API_KEY	Your Groq key
SECRET_KEY	Any long random string
DATABASE_URL	sqlite:///./nyaay_ai.db  (or PostgreSQL URL)

Frontend → Vercel
Framework	Next.js (auto-detected)
Root Directory	frontend
NEXT_PUBLIC_API_URL	https://nyaayai-tptx.onrender.com
NEXT_PUBLIC_USE_MOCK	false

After Vercel deploys, add your Vercel URL to the allow_origins list in backend/app/main.py, then push to trigger a Render redeploy.

🐛  Common Issues
Problem	Fix
Failed to fetch	Backend is sleeping — open the Render URL directly, wait 30s, then try again
CORS error	Add your Vercel URL to allow_origins in backend/app/main.py and redeploy
DATABASE_URL error	Add DATABASE_URL=sqlite:///./nyaay_ai.db in Render environment variables
Tesseract not found (local)	Install: brew install tesseract (macOS) or sudo apt install tesseract-ocr
Vercel build fails	Ensure Root Directory is set to frontend in Vercel project settings
Groq API errors	Verify GROQ_API_KEY is set in Render env and starts with gsk_
